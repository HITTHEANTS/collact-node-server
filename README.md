## Description

Collact API server (using [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository)

## Development tool pre Requirements

```
node 14.x
postgresql 13.1
```

## Installation

```bash
$ npm install
```

## Pre Requirements

```
You should set .env file for reference .env.EXAMPLE file.
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## How to docker deployment

```bash
$ make run_prod

go to browser && http://localhost:80/api
```
